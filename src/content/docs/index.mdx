## The shift from senior to principal

A senior engineer names the **decision**. A principal makes the decision in a way a working system can survive when things break — and ships with eyes open.

Before reaching for an alternative or rejecting one, say out loud: **the choice → the alternative rejected → the trade-off accepted → the failure mode at 10×.** Name the property you need first, then the tech.
